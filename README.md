# ddz_net_client

## 项目简介

本项目为斗地主的客户端，采用QT5.15.2编写，涉及QT的哈希，多线程QRunnable，Multimedia模块，QSS美化，openssl，protobuf等。并使用innosetup进行打包。可以进行单机或联机。服务端详情：[feibo888/ddz_net_server: 可以单机或联机的斗地主](https://github.com/feibo888/ddz_net_server)
